## 来源
[jQuery Raty](https://github.com/wbotelhos/raty)

## 使用
```
js:
 angular.module('app'), ['raty'];

html:
 <div raty ng-model="xxx"></div>
```

## 参数
```
ng-model {Scope}           scope
readonly {Boolean} [false] 是否允许编辑
```
